Iris --
This project took inspiration from my bedroom, and is an amalgamation of what --
that represents into a webpage. It's pretty much the first thing I had to code --
all by myself and I've improved the code a lot in this version. --

Diamond --
This project is actually a picture of my eldest dog Chief. My original code was --
just circles for pixels so I decided to spice it up by adding some more shapes. --
I think it turned out well, as the diamond pattern that is formed is pretty cool. --

Celestial --
This project is an abstract clock that uses the first three planets and the sun --
to keep track of time. I like that at first glance you can't even tell that it is --
a clock. I've updated the code so that its formatted a lot nicer. --

Palindrome --
This project may seem simple but there is a lot of symbolism. The words themselves are --
a palindrome, and its formatted so the pages stack on top of eachother. It plays into --
the idea that life is in fact never odd or even, its often grey. --
